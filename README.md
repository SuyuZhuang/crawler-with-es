# 多线程爬虫和ES数据分析实战

## 1 算法
1. 从待处理链接池中拿一个链接
2. 如果已经处理过了，拿下一个
3. 如果没有处理过，检查是否是我们想要的链接
4. 如果是我们想要的，根据链接拿到page中所有其他链接，放入待处理池子中
5. 该页面如果是新闻链接，将新闻相关的标题、内容存储下来
6. 回到第1步
